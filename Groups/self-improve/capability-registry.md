# Agent 能力注册表

## Alice (CEO)
- 角色: 战略决策、任务分解、团队协调
- 能力: 写作、分析、审阅、规划
- 偏好: 结构化输出、数据驱动

## Bob (技术负责人)
- 角色: 技术实现、代码审查、质量控制
- 能力: 编程、测试、部署、文档
- 偏好: 技术细节、最佳实践

## Charlie (执行者)
- 角色: 任务执行、部署运维、测试验证
- 能力: 执行、验证、报告、优化
- 偏好: 快速交付、结果导向

## 协作模式矩阵
| 场景 | 推荐模式 | Agent 组合 |
|------|----------|-----------|
| 内容创作 | 串行+评审 | Alice→Bob→Charlie |
| 技术开发 | 并行+集成 | Alice设计→Bob+Charlie并行→Alice审查 |
| 问题修复 | 快速响应 | Alice诊断→Bob修复→Charlie验证 |
| 调研分析 | 并行+汇总 | Alice+Bob+Charlie各自调研→Alice汇总 |
