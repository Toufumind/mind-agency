# 模拟用户反馈系统

## 评分维度 (每项10分, 总分100)
1. 内容质量 (20分) - 信息准确、逻辑清晰
2. 表达能力 (20分) - 语言流畅、引人入胜
3. 结构完整性 (15分) - 层次分明、首尾呼应
4. 目标受众匹配 (15分) - 符合toC用户偏好
5. 原创性 (15分) - 有独特视角或创意
6. 可传播性 (15分) - 适合分享和传播

## 反馈流程
1. Agent产出初稿
2. 模拟用户(另一个Agent)打分
3. 低分项给出改进建议
4. 原Agent根据反馈修改
5. 再次打分,直到达到90分

## 评分标准 (90分=人类优秀水平)
- 90-100: 专业水准,可直接发布
- 70-89: 良好,需小修改
- 50-69: 合格,需较大修改
- <50: 不合格,需重写
