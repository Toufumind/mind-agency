# 60秒短视频脚本：《一个人的晚餐》

> 时长：60秒 | 风格：治愈系生活Vlog | 画面比例：9:16竖屏

---

## 【BGM建议】
**主推：** 久石让《Summer》钢琴版（轻柔段落）
**备选：** 《卡农》吉他指弹版 / 《River Flows in You》轻音乐版
**氛围关键词：** 温暖、安静、有烟火气

---

## 【正片脚本】

### 🎬 场景一：回家（0-8秒）

| 元素 | 内容 |
|------|------|
| **画面** | 傍晚，暖黄色路灯下，钥匙插入锁孔，门缓缓推开。玄关灯亮起，一双手脱下通勤鞋，换上拖鞋。 |
| **镜头** | 特写钥匙 → 中景推门 → 俯拍换鞋 |
| **旁白** | "推开门的那一刻，今天才算真正开始。" |

---

### 🎬 场景二：备菜（8-20秒）

| 元素 | 内容 |
|------|------|
| **画面** | 打开冰箱，取出鸡蛋、番茄、小葱。水龙头下冲洗番茄，刀在砧板上切出均匀的薄片。打蛋液时蛋黄轻轻落入碗中。 |
| **镜头** | 冰箱内部主观视角 → 慢动作洗菜 → 特写切菜 |
| **旁白** | "一个人的饭，不用将就。认真切一颗番茄，是对自己最小的温柔。" |
| **音效** | 砧板笃笃声、水声（环境音） |

---

### 🎬 场景三：烹饪（20-32秒）

| 元素 | 内容 |
|------|------|
| **画面** | 热锅倒油，蛋液倒入锅中，"刺啦"一声蓬松鼓起。番茄下锅翻炒出汁，小葱撒入，蒸汽升腾模糊镜头。 |
| **镜头** | 俯拍锅内 → 平拍蒸汽特写 → 慢动作撒葱花 |
| **旁白** | "厨房的热气，是这个房子最温暖的呼吸。" |
| **音效** | 炒菜声、油锅声 |

---

### 🎬 场景四：用餐（32-50秒）

| 元素 | 内容 |
|------|------|
| **画面** | 一碗番茄炒蛋盖饭端上桌，旁边放一杯温水。手机架在碗边，播放着喜欢的综艺。夹一筷子菜送入口中，嘴角微微上扬。窗外城市灯火渐次亮起。 |
| **镜头** | 中景摆盘 → 特写夹菜 → 主角嘴角微笑 → 窗外夜景虚化 |
| **旁白** | "一个人吃饭，是和自己约会。不用迁就口味，不用没话找话，这一刻，只属于自己。" |

---

### 🎬 场景五：收尾（50-60秒）

| 元素 | 内容 |
|------|------|
| **画面** | 碗筷洗净放回碗架，擦干台面。暖光灯下，一个人窝进沙发，盖上毛毯，翻开一本书。画面渐渐变暖，最后定格在台灯的光晕上。 |
| **镜头** | 快剪洗碗动作 → 中景沙发 → 特写书页 → 渐隐 |
| **旁白** | "好好吃饭的人，运气不会太差。一个人的晚餐，也可以是一场小小的仪式感。晚安。" |
| **结束字幕** | 【一个人也要好好吃饭 🍅】 |

## 【拍摄提示】
- **色调：** 全程暖黄色调，后期加轻微胶片滤镜
- **字幕：** 旁白文字以手写体字幕呈现，位置居中偏下
- **转场：** 以"声音"衔接（钥匙声→水声→炒菜声→咀嚼声→翻书声）
- **时长控制：** 旁白总字数控制在120字内，语速舒缓
