## 任务详情

### 目标
优化前端性能，将首屏加载时间降至 1.5s 以内。

### 具体事项
1. **首屏加载分析** — 使用 Lighthouse 或 Web Vitals 测量当前性能基线
2. **SSE 连接复用** — 检查 EventSource 实例是否可跨组件共享，避免重复连接
3. **代码分割** — 对 Markdown 渲染、图表等重型组件使用 `dynamic(() => import(...))` 懒加载
4. **资源优化** — Tailwind CSS purging、图片压缩、字体子集化
5. **缓存策略** — 为静态资源设置合理的 Cache-Control

### 验收标准
- Lighthouse Performance Score ≥ 90
- FCP < 1.0s, LCP < 1.5s
- SSE 连接数 ≤ 1 个
