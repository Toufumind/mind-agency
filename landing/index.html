Quick Start

5 分钟上手

目前支持 Claude 和 DeepSeek，更多模型接入中。

01

安装

下载 exe 双击运行，或从源码

$ git clone github.com/Toufumind/mind-agency
$ cd mind-agency && npm install
$ npm run dev
02

配置

首次打开会引导你填 API Key

Settings → API Key → Save
03

建团队

系统自带 Alice/Bob/Charlie，也能自己建

Agents/ 目录下加 config.json
04

跑起来

建群组，定义工作流，点运行

$ curl localhost:3001/workflows/run
FAQ

常见问题

多个 AI 怎么协作？谁听谁的？+
你定义规则，AI 按规则分工。Alice 写完 Bob 自动审，有分歧投票，关键步骤等你批准。
需要花钱吗？+
软件免费开源。需要 AI 模型的 API Key，DeepSeek 几毛钱一天。
不会写代码能用吗？+
下载 exe 双击运行，跟着向导配置。系统自带示例 Agent，开箱即用。
数据会被上传吗？+
不会。数据全在本地，只有 AI 对话会发到你配置的 API。
支持 macOS / Linux 吗？+
当前 exe 仅支持 Windows。macOS / Linux 在路线图中，也可从源码运行。
