Features

你需要的一切

不是单个 AI，而是一个完整的协作系统。

01

多 Agent 协作

创建任意数量的 Agent，各有角色和性格。Alice 写代码，Bob 审查，Charlie 测试。有分歧投票决定。

02

工作流引擎

YAML 定义步骤和依赖。失败自动重试，崩溃从检查点恢复。支持人工审批门。

03

三层记忆

长期记忆、项目记忆、会话记忆。Agent 记住每次协作的经验，不会重复犯错。

04

共识投票

AND / OR / 阈值投票 + 对抗审查。多轮辩论达成共识。

05

完整审计

每个 Agent 操作都被记录和追踪。从邮件到群组聊天，全部可追溯。

06

多主题

5 种主题风格：Notion、极简白、暖木、深空、Nord。支持多语言。

常见问题

多个 AI 怎么协作？
你定义规则，AI 按规则分工。Alice 写完 Bob 自动审，有分歧投票，关键步骤等你批准。
需要花钱吗？
软件免费开源。需要 AI 模型的 API Key，DeepSeek 几毛钱一天。
数据会被上传吗？
不会。数据全在本地，只有 AI 对话会发到你配置的 API。
支持 macOS / Linux 吗？
当前桌面客户端仅支持 Windows。macOS / Linux 可从源码运行。
和 CrewAI / AutoGen 有什么区别？
CrewAI 是框架，Mind Agency 是完整产品——可视化界面、共识投票、崩溃恢复，开箱即用。
Agent 之间怎么通信？
三种方式：群组聊天（@提及自动响应）、点对点邮件、MCP 工具调用。
