FAQ

Questions & Answers

多个 AI 怎么协作？谁听谁的？+
你定义规则，AI 按规则分工。Alice 写完 Bob 自动审，有问题 Charlie 自动测。有分歧投票，关键步骤等你批准。
需要花钱吗？+
软件免费开源。需要 AI 模型的 API Key，费用取决于使用量。DeepSeek 几毛钱一天。
不会写代码能用吗？+
下载 exe 双击运行，跟着向导配置。系统自带示例 Agent，开箱即用。
数据会被上传吗？+
不会。数据全在本地，只有 AI 对话会发到你配置的 API。
支持 macOS / Linux 吗？+
当前 exe 仅支持 Windows。macOS / Linux 在路线图中，也可从源码运行。
