Quick Start

3 分钟跑起来

目前支持 Claude 和 DeepSeek，更多模型接入中。

01

克隆仓库

需要 Node.js 18+

$ git clone https://github.com/Toufumind/mind-agency $ cd mind-agency && npm install
02

启动服务

会自动打开浏览器

$ npm run dev ✓ Ready on http://localhost:3000
03

配置 API Key

首次打开有向导引导

Settings → API Key → DeepSeek 费用：几毛钱一天
04

创建团队

系统自带 Alice/Bob/Charlie

在 Agents/ 下加 config.json 或让 Agent 自己创建新成员
FAQ

常见问题

多个 AI 怎么协作？+
你定义规则，AI 按规则分工。Alice 写完 Bob 自动审，有分歧投票，关键步骤等你批准。
需要花钱吗？+
软件免费开源。需要 AI 模型的 API Key，DeepSeek 几毛钱一天。
数据会被上传吗？+
不会。数据全在本地，只有 AI 对话会发到你配置的 API。
支持 macOS / Linux 吗？+
当前 exe 仅支持 Windows。macOS / Linux 在路线图中，也可从源码运行。
和 CrewAI / AutoGen 有什么区别？+
CrewAI 是框架，Mind Agency 是完整产品——可视化界面、共识投票、崩溃恢复、审计日志，开箱即用。
