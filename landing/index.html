协作

分工明确，
像真正的团队。

Alice 写代码，Bob 审查，Charlie 测试。每个人有自己的专长，通过群聊和邮件协作。有分歧？投票决定。

  • 角色分工：每个 Agent 有自己的职责和专长

自动化

流水线定义好，
团队自动跑。

  • 步骤编排：A 做完 B 才能开始，自动流转

FAQ

Questions & Answers

多个 AI 怎么协作？谁听谁的？ +
每个 Agent 有自己的角色和专长。Alice 写代码，Bob 审查，Charlie 测试。有分歧时投票决定。关键决策可以暂停等你批准。就像人类团队一样，有分工也有民主。
需要花钱吗？ +
软件本身免费开源。需要 AI 模型的 API Key（Claude / DeepSeek / GPT-4o），费用取决于你的使用量。DeepSeek 价格很低，几毛钱一天。
我不会写代码，能用吗？ +
可以。下载 exe 双击运行，跟着向导配置 API Key。系统自带三个 Agent（Alice、Bob、Charlie），开箱即用。创建群组和工作流都可以在界面里操作，不需要写代码。
我的数据会被上传吗？ +
不会。所有数据存储在你本地电脑上，不上传任何地方。只有 AI 对话会发送到你配置的 API（Claude/DeepSeek/GPT），你可以随时查看。
支持 macOS / Linux 吗？ +
当前 exe 仅支持 Windows。macOS 和 Linux 在路线图中。也可以从源码在任何平台运行。
